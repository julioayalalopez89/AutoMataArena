# AutoMataArena

AutoMataArena is the real-time combat engine of the AutoMataSoul universe.
It enables remote-controlled and AI-driven roboces to move, interact,
and fight over the internet with low-latency synchronization.

## Features
- Real-time robot movement over the internet
- AI and human-controlled roboces
- Modular control system for physical and digital robots
- Authoritative state synchronization
- Designed for scalability and future arenas
